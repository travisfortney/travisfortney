# Travis Fortney

I work in support operations and like building small tools that make repetitive work easier to see, check, or finish.

## Current Little Projects

- [support-health-formula](https://github.com/travisfortney/support-health-formula) - a synthetic support ops scoring model with sample data and tests
- [done-ding](https://github.com/travisfortney/done-ding) - a tiny macOS chime for long-running terminal task alerts
- [now-playing-notifier](https://github.com/travisfortney/now-playing-notifier) - macOS Spotify notifications with a local-device filter for Spotify Connect noise
- [larry-david-desktop-pet](https://github.com/travisfortney/larry-david-desktop-pet) - a small desktop pet package based on my cat, Larry David

I like boring checks, readable docs, and small utilities that remove one tiny source of future irritation.
